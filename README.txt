epikriz.info — структурований старт
Папки розділів:
- /mri — для МРТ (PDF/JPG/PNG)
- /uzi — для УЗД
- /reports — для виписок/висновків
- /msek — для пакету МСЕК (рекомендовано закрити Cloudflare Access)
- /files — будь-які додаткові файли
- /assets — статичні ресурси

Як додати перший документ у галерею:
1) Покладіть свій файл у /reports (наприклад, reports/2025-epikriz.pdf).
2) У index.html в <div id="items"> замініть приклад картки на свій і шлях у data-open.
3) Закомітьте зміни — сайт оновиться автоматично.
